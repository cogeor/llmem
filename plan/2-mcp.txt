# MCP Server Implementation Plan (Architectural Helper)
# Component: src/mcp/

================================================================================
## TOOLS
================================================================================

### 1. get_artifacts
**Description**: Get architectural insights for a folder. Returns file signatures and prompts for a summary.
**Args**:
- `path`: string (Folder path)
- `recursive`: boolean (default false)

**Behavior**:
1. Calls `artifact.ensureArtifacts(path)`.
2. Aggregates signatures.
3. Returns `prompt_ready` response instructing Host LLM to summarize the folder.
4. Sets callback to `store_folder_summary`.

### 2. store_folder_summary
**Description**: Save the summary generated by the Host LLM.
**Args**:
- `path`: string (Folder path)
- `summary`: string (Markdown content)

**Behavior**:
1. Calls `artifact.saveFolderSummary(path, summary)`.
2. Returns success.

================================================================================
## INTERFACES
================================================================================

```typescript
const GetArtifactsSchema = z.object({
  path: z.string().describe("Folder path to analyze"),
  recursive: z.boolean().optional().default(false),
});

const StoreFolderSummarySchema = z.object({
  path: z.string().describe("Folder path"),
  summary: z.string().describe("Generated summary"),
});
```
